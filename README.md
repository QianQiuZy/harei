### 运行步骤（如果你想自己调试我默认会python的喵~）

1. 克隆仓库：
   ```bash
   git clone https://github.com/QianQiuZy/harei.cn
   ```

2. 配置环境：
   ```bash
   pip install pdm
   pdm install
   ```

4. 运作项目：
   ```bash
   pdm run server/run.py
   ```
